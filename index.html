Sobre

Este é o quarto PyCaxias e o objetivo é movimentar a serra gaúcha! A região já sediou uma edição da Python Brasil e Python Sul, também é o berço de grandes nomes que contribuíram para que a comunidade Python tenha alcançado seu nível atual.
